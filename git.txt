🔄 How to Update Your Site After Making Changes
It's super easy — just 3 Git commands:

bash

git add -A
git commit -m "Description of your changes"
git push

That's it! Render automatically detects the push and redeploys your site (takes ~2-3 minutes).

Example Workflow:
Make changes to your code locally
Test locally (npm start)
When ready, run:
bash

git add -A
git commit -m "Fixed reservation bug"
git push

Render auto-deploys — check the Render dashboard to see progress
Your live site updates automatically!
